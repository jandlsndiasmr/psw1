Site de PSW

Bem-vindos ao Site da Turma

Este site foi criado como um projeto da disciplina de Programação de Sistemas Web I (PSW1) do IF Baiano, Campus Guanambi.

O objetivo é oferecer um espaço onde os alunos possam anexar seus projetos. Cada link aqui representa um sistema específico desenvolvido pelos alunos, refletindo criatividade e aplicando conceitos de HTML, CSS, PHP, e banco de dados.

Projetos Disponíveis:

  • Buffet Online - Um sistema de gerenciamento de pedidos e eventos para buffets, facilitando o planejamento e a organização de serviços de alimentação.
  • Cineview - Uma plataforma de exibição de informações e críticas de filmes.
  • Livraria Virtual - Um sistema de e-commerce voltado para livros.
  • NeuroBiblioteca - Uma biblioteca digital com foco em neurociências.
  • Sistema de Doação - Plataforma para facilitar doações, conectando doadores a instituições de caridade.
  • Sistema de Gravadora - Um site voltado para gerenciamento de gravadoras e artistas.
  • Sistema de Papelaria - Plataforma para gerenciamento de papelarias, permitindo consulta e gestão de materiais escolares e de escritório.
  • Sistema de Esporte - Um site para gestão de marcas e esportes diversos.
  • Sistema de Restaurante - Um sistema de pedidos para restaurantes.

Esses projetos foram desenvolvidos ao longo do semestre como prática de programação. Acesse os links e confira o trabalho da turma de PSW1!
